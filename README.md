# Neemasree's Portfolio Website

A professional portfolio website built with HTML, CSS, and JavaScript showcasing skills, projects, and experience.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Interactive Navigation**: Smooth scrolling and active section highlighting
- **Skills Showcase**: Animated skill bars and categorized expertise
- **Project Portfolio**: Featured projects with descriptions and tech stacks
- **Downloadable Resume**: Printable resume page that can be saved as PDF
- **Contact Form**: Functional contact form with validation
- **Professional Sections**:
  - Home/Hero section with introduction
  - About page with personal information and stats
  - Skills page with technical expertise
  - Projects showcase with live demos and GitHub links
  - Resume download functionality
  - Contact page with form and social links

## File Structure

```
portfolio/
├── index.html          # Main portfolio page
├── styles.css          # All styling and responsive design
├── script.js           # Interactive functionality
├── resume.html         # Printable resume page
└── README.md          # This file
```

## Getting Started

1. Open `index.html` in your web browser
2. Navigate through different sections using the navigation menu
3. Customize the content with your own information
4. Replace placeholder images with your actual photos
5. Update contact information and social media links
6. Add your real project links and GitHub repositories

## Customization

### Personal Information
- Update name, title, and description in the hero section
- Replace contact details in the contact section
- Add your actual social media links
- Update the about section with your personal story

### Skills
- Modify skill categories and items in the skills section
- Adjust skill bar percentages to reflect your expertise
- Add or remove technologies based on your experience

### Projects
- Replace placeholder projects with your actual work
- Add real project images (replace image placeholders)
- Update project descriptions and technology stacks
- Add live demo and GitHub repository links

### Resume
- Edit `resume.html` with your actual work experience
- Update education, certifications, and achievements
- Modify the skills list to match your expertise
- Print or save as PDF for download functionality
